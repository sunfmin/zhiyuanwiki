# 上线范围：黑龙江先行（物理+历史），数据模型按省份×科类×年份建键

上线范围限定为**黑龙江省**的**物理类+历史类**，因为我们手上完整的历年数据（招生计划/一分一段/投档线/专业录取分数线 2020–2026）只覆盖黑龙江，且方法论也是在黑龙江物理类场景下打磨出来的。

但数据模型从第一天起就把 **`省份 × 科类 × 年份`** 当作一等复合键，使"加省份"只是加数据、不是改架构。明确**不**在上线时做多省份——其他省份需另行采集各省考试院数据且方法论未经验证，会拖慢上线并引入未核验的数据。

上线前须核实：历史类的四套数据是否与物理类同样齐全（一分一段历史类已确认存在，投档线/专业线/招生计划的历史类完整度待核）。
