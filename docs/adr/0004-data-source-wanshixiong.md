# 数据来源：采用万师兄第三方整理数据公开发布

叶子页最丰富的数据（逐专业录取最低分/最低位次/最高分/选科要求，2020–2025）来自**万师兄·高考志愿填报大数据**——一份**第三方付费整理版，非省考试院直发**。决定在公开网站上采用其全部数据，以换取最完整的院校×专业叶子页。

**已知风险**：录取分数线/位次这类客观事实本身一般不受版权保护，但他人整理成品的具体编排可能受保护，直接公开转载存在版权/转载风险；且数据未经省考试院直接背书，可能有录入误差。

**缓解措施**（须落实）：① 全站显著标注数据来源；② 加"仅供参考、以官方及院校最新公布为准"免责声明；③ 关键档定稿前抽样比对 lzk.hl.cn 官方在线查询系统；④ 尽量从官方源（投档线、一分一段、招生计划）核验并以本站自有结构重新生成，而非原样搬运成品表格文件。
