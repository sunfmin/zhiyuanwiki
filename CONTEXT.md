# 高考志愿数据 Wiki（黑龙江先行）

帮助高考考生用历年官方数据选择院校与专业的**公开 wiki**。本文件是术语表（统一用词），不放具体方案与实现细节——方案性的关键决定记在 `docs/adr/`。

## Language — 录取数据

**位次**：
考生在本类别（物理类/历史类）内的全省累计排名。报志愿的**核心参照**——比分数稳定，因为题目难易每年浮动，但"排第几名能上哪所学校"相对恒定。本站一切定位、筛选、走势均以位次为准。
_Avoid_：名次、排名、分数（分数是表象，位次才是依据）

**一分一段表**：
省招生考试院发布的"每个分数及其以上累计人数"对照表，用来把分数换算成位次。每个（省份×科类×年份）一张。

**等效位次**：
把往年某校某专业的录取位次，放到今年来看大约相当于多少分/什么名次，从而跨年份比较"今年够不够得着"。本站做跨年比较的唯一正确方法。
_注_：位次是唯一基准——排序、冲稳保把握分档（按 V/R 比值）一律以位次为准。**等效分**（由等效位次经最新一分一段 `rankToScore` 反查）仅作**辅助展示**，帮考生"感知"差距，永远标「约/≈」、绝不作排序或分档依据。

**投档线**：
某**院校专业组**当年的投档最低分及其对应位次（平行志愿按组投档）。来源为官方公开数据。

**专业录取分数线**：
某**院校×专业**当年的录取数据——最低分、最低位次、最高分、选科要求。是叶子页最丰富的数据。
_Avoid_：专业分数线（同义，站内统一用"专业录取分数线"）

**招生计划**：
某年某院校专业组下逐专业的计划招生人数、学制、学费、选科要求、专业全称/备注。是"今年能填什么"的真相来源。

## Language — 主体实体

**院校**：
一所学校，由稳定的校代码标识。**本站的主干稳定实体**——每所院校一个常驻页面，聚合其历年投档线/位次走势、招生计划与开设专业。
_Avoid_：大学、学校、高校（站内统一用"院校"）

**专业**：
一个学科方向（如"计算机科学与技术"）。概念跨年稳定，可跨校横向对比。

**院校×专业**：
某所院校开设的某个专业。**本站的数据叶子页**——承载该(院校,专业)的历年专业录取分数线与选科要求走势。

**院校专业组**：
平行志愿的实际填报单位：一所院校内若干专业打包成一组，共享同一套选科要求与"是否服从调剂"。**组号逐年变（是单年产物，不是跨年稳定实体）**，故本站只把它作为"某一年的报考视图"呈现在院校页之下，不做永久可互链的实体页。
_Avoid_：专业组、招生组（站内统一用"院校专业组"）

## Language — 填报与定位

**平行志愿**：
同一批次可填多个志愿，按"位次优先、遵循志愿顺序"投档，每人只投一次。意味着志愿表的**顺序**很关键。

**冲 / 稳 / 保**：
按录取把握给每个候选**院校×专业**贴的标签，依据访客位次 V 与该专业等效录取位次 R 的比值——**保**（V/R 明显小，稳稳够得着）、**稳**（V/R≈1，与本人相当）、**冲**（V/R 略大，录取位次略好于本人、搏一把）。**是标签、不是定额**：每档只显示真正落入该比值区间的条目，按"最贴近本人位次"在前排序，**绝不靠拉远的条目凑数**（密集时软上限 + 展开更多，稀疏时有几条显几条）。具体阈值见 ADR-0010。

**够不着 / 过保**：
冲稳保两端之外的**远档**——**够不着**＝比冲更难（R 远好于 V，基本搏不到）；**过保**＝比保更易（R 远差于 V，白白浪费位次）。本站不把它们误标成冲/保，而是放进"仅供参考"折叠区（置灰、显式标记）。当真实冲/保不足时，用它们把冲列、保列各**补齐到约 100 条**——凑数只发生在这个被降级、被标记的远档区，**主档永远只装真档**。

**服从专业调剂 / 组内调剂**：
所报专业满额时，是否同意被调到**同一院校专业组内**的其他专业（只在组内，不跨组）。勾"服从"大幅降低退档风险，代价是可能被调到不喜欢的专业——故选组时要看组内有没有绝对不想读的专业。

**选科 / 物理类 / 历史类**：
新高考 3+1+2（**黑龙江**模式）下首选物理或历史的考生类别。两类**分开排名、分开划线、分开录取**，同一分数含金量不同；位次、一分一段、投档线均按类别分别计算。
_Avoid_：理科/文科（旧高考叫法，已不准确）
_注_：物理类/历史类是 3+1+2 省份（黑龙江）特有；浙江是 3+3，单科类「综合」，见下「浙江」节。

**选科要求**：
某院校专业组/专业对再选科目的限定（如"不限""化学""化学和生物"）。决定一个考生的选科组合能不能填某个组。

**访客位次**：
访客在浏览器本地输入的自己的位次（及科类/选科），仅在客户端用于筛选与冲稳保分档，**不上传、不存储**。

## Language — 浙江（综合 / 专业平行志愿）

浙江是 3+3 新高考，模型与黑龙江（3+1+2）不同。详见 ADR-0009。

**综合**：
浙江的**唯一科类**——不分物理/历史，全省一套排名、一套一分一段。本站对浙江的位次、定位、走势都按「综合」单科类计算。
_Avoid_：物理类/历史类（那是黑龙江的叫法，浙江不适用）

**选考（7选3）**：
浙江考生在物理、化学、生物、政治、历史、地理、技术 7 科中**选 3 科**作为选考科目；专业的选科要求据此判定能否填报。
_注_：「技术」是浙江特有选考科目；要求格式如「物理、化学(2科必选)」「物化生(3科必选)」「物/化/生(3选1)」「思想政治必选」。

**普通类一段 / 二段**：
浙江普通类按分数分**段**录取：一段线以上为一段，其下到二段线为二段（命名逐年变，旧年份叫「平行录取一段/二段」）。一段≈竞争性本科段。本站收录 一段 + 二段 + 提前批 的全量普通类。
_Avoid_：本科批/专科批（浙江已合并为普通类分段）

**专业平行志愿**：
浙江的填报单位是**院校×专业**——每个院校的每个专业都是独立投档单位，按"位次优先、遵循志愿顺序"投档，**无组内调剂**（不存在院校专业组）。
_Avoid_：院校专业组/服从调剂（那是黑龙江等省的规则，浙江不适用）

## Language — 院校 / 专业属性（筛选维度）

**学科门类（门类）**：
专业的最高层学科归类，共 12 类（工学、理学、医学、管理学、经济学、文学、法学、教育学、农学、艺术学、历史学、哲学）。本站把"专业大类"过滤定在门类这一级。
_注_：官方新科类数据不带门类列，本站按归一化专业名自建门类映射，未命中归"其他"。

**院校层次**：
院校的国家工程标签——**985 / 211 / 双一流**（三者大致 985 ⊂ 211 ⊂ 双一流）。来自第三方整理的院校标签，按校名挂接。
_Avoid_：重点大学、名校（站内统一用具体标签）

**办学性质**：
院校是**公办**还是**民办**（含独立学院）。影响学费量级，家长高度关注。

**学校类别**：
院校的类型——综合 / 理工 / 师范 / 医药 / 财经 / 农林 / 政法 / 艺术 / 体育 / 民族 / 语言 等。

**城市层级**：
院校所在城市的分级（一线 / 新一线 / 二线 …）。来源表无此列，本站按静态"城市→层级"映射判定。

**中外合作**：
中外合作办学专业/项目，学费通常远高于普通专业。按专业名/全称/备注含"中外""合作办学"判定。
